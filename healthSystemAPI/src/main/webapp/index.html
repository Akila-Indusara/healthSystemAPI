Health System API

Endpoints

Person

    @GET

  • ...rest/people - get all People

    • @GET

  • ...rest/people/byID/{PersonID} - get Person by ID

    • @GET

  • ...rest/people/byName/{PersonName} - get Person by Name

    • @POST

  • ...rest/people - add Person

    • Example JSON body:

    {
        "name": "testPerson",
        "age": 18,
        "contactNumber": 456,
        "address": "test",
        "id": "Per01"
}

    @PUT

  • ...rest/people/{PersonID} - update Person

    • Example JSON body:

    {
        "name": "testPerson",
        "age": 18,
        "contactNumber": 456,
        "address": "testUpdated",
        "id": "Per01"
}

    @DELETE

  • ...rest/people/{PersonID} - delete Person by ID

Medical Records

    @GET

  • ...rest/medical_records - get all Medical Records

    • @GET

  • ...rest/medical_records/{recordID} - get Medical Records by ID

    • @GET

  • ...rest/medical_records/byPatient/{PatientID} - get Medical Records by Patient ID

    • @Post

  • ...rest/medical_records/{PatientID} - add Medical record belonging to a existing patient

    • Example JSON body:

    {
  "medical_RecordID": "MR123456",
  "diagnosesAndtreatments": {
    "diagnosis1": "treatment1",
    "diagnosis2": "treatment2"
  },
  "currentMedications": [
    "Medication1",
    "Medication2"
  ],
  "allergies": [
    "Allergy1",
    "Allergy2"
  ],
  "medicalConditions": [
    "Condition1",
    "Condition2"
  ]
}

    @PUT

  • ...rest/medical_records/{recordID} - Update Medical record by ID

    • Example JSON body:

    {
  "medical_RecordID": "MR123456",
  "diagnosesAndtreatments": {
    "diagnosis1": "treatment1Updated",
    "diagnosis2": "treatment2Updated"
  },
  "currentMedications": [
    "Medication1Updated",
    "Medication2"
  ],
  "allergies": [
    "Allergy1Updated",
    "Allergy2"
  ],
  "medicalConditions": [
    "Condition1Updated",
    "Condition2"
  ]
}

    @Delete

  • ...rest/medical_records/{recordID} - Update Medical record by ID
